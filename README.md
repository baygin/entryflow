# ekşisözlük entryflow

ekşi sözlük'te f5'e basmadan yeni entry'leri getiren chrome eklentisi.

sayfayı yenilemeden başlığa düşen fresh entry'ler otomatik geliyor, yeşil etiketle "yeni" diye bağırıyor, scroll da kendisi yapıyor. dark mode destekli.

## kurulum

chrome web store'dan indir: https://chrome.google.com/webstore/detail/eksi-flow/abcdefghijklmnopqrstuvwxyzabcdef

## kullanım

- başlık sayfasına gir, sağ altta panel belirir
- otomatik güncellemeyi başlat (varsayılan 30sn aralıkla kontrol eder)
- yeni entry gelince otomatik eklenir ve scroll yapar
- manuel kontrol de var, acele edenlere

ayarlardan kontrol süresini değiştirebilirsin (10-300sn arası).

## not

ekşi sözlük'ün resmi ürünü değil, sadece f5 köleliğinden kurtuluş aracı.
